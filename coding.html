// animated text vars

const headerText = 'My name is Lamorna Read';
const paraText = 'I am an aspiring web developer with a great passion for being creative and using code to do it.';
let headerSpeed = 80;
let paraSpeed = 60;
let headerLength = 0;
let paraLength = 0;

// animated text function 

function typeWriterHeader() {
    if (headerLength < headerLength.length) {
        document.getElementById('type-header').innerHTML += headerText.charAt(headerLength);
        headerLength++;
        setTimeout(typeWriterHeader, headerSpeed);
    }
}
function typeWriterPara() {
    if (paraLength < paraText.length) {
        document.getElementById('type-para').innerHTML += paraText.charAt(paraLength);
        paraLength++;
        setTimeout(typeWriterPara, paraSpeed);
    }
}

function typeWriter() {
    typeWriterHeader();
    setTimeout(typeWriterPara, 3000);
}

typeWriter();

Typewriter function

I created this function using JavaScript. It's used to create the typing animation for the homepage banner.

It does this by adding the text content with JavaScript and adds one letter at a time until all the content is added

Having two seperate functions for the header and footer help me to be able to put them both into one and create delay between them for a more pleasent visual effect.

It also needs to be on its own file and only applied to page that needs it so it doesnt stop the rest of your code running.

                                
                                    
JavaScript

//hamburger vars

const hamburger = document.querySelector('.hamburger');
const navSmall = document.querySelector('.small-nav');
const navLink = document.querySelectorAll('.nav-li');

//hamburger event and function

hamburger.addEventListener("click", mobileMenu);

function mobileMenu() {
    hamburger.classList.toggle('active');
    navSmall.classList.toggle('active');
};

navLink.forEach(n => n.addEventListener('click', closeMenu));

function closeMenu() {
    hamburger.classList.remove('active');
    navSmall.classList.remove('active');
}

Hamburger Button

The hamburger button is a well known design used by the web developer community. It is commonly used for smaller screen design to hide our navigation bars, for improved flow and functionality.

I used JavaScript to create and tell the page how it works. I gave it the instructions that it needed to open and close the navigation menu on click.

Once I had done this I added the CSS to design how it looked and sat on the page. Each line of CSS used targets what I want to design and makes it look the way I want it too.
